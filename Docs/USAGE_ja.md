# AO Curvature Baker 使い方ガイド

## 1. ツール概要
AO Curvature Bakerは、3Dモデルに対して「Ambient Occlusion (AO / 影)」と「Curvature (曲率 / エッジ)」を計算し、テクスチャとして保存するためのUnityエディタ拡張ツールです。
GPUを利用した高速な計算により、モデルの隅に溜まる影や、角の擦れを表現するためのマスクを簡単に作成できます。

## 2. 使用する流れ
1.  **モデルの登録**: `TARGET` セクションに、影を付けたいオブジェクトをドラッグ＆ドロップします。
2.  **設定の調整**: 必要に応じて、影の濃さや解像度などを調整します（基本はそのままでも動作します）。
3.  **ベイク実行**: 下部の `Bake Now`（ベイク実行）ボタンを押します。
4.  **確認**: 計算が終わると、モデルと同じフォルダ（または指定したフォルダ）にテクスチャが生成されます。

## 3. 各機能の説明
### TARGET（ターゲット）
*   **Add Target**: ここに登録したオブジェクトに対してテクスチャが作成されます。MeshFilter または SkinnedMeshRenderer が必要です。

### OCCLUDER MESHES（影を落とす物）
*   **Add Occluder**: ターゲットに対して影を落とすための追加オブジェクトを登録できます。背景や他の小物などがこれに当たります。

### BASIC AO SETTINGS（基本設定）
*   **Self Occlusion**: 自分自身が自分に落とす影を計算します。
*   **Mutual Occlusion**: 登録したターゲット同士が互いに落とす影を計算します。
*   **Low Resource Mode**: ベイク中にPCが固まる場合はこれをオンにしてください。

### ADVANCED SETTINGS（詳細設定）
*   **影の細かさ (Ray Count)**: 影のザラつきに影響します。多いほどきれいです。
*   **影の広がり (Max Distance)**: どのくらい遠くの物まで影として考慮するかです。

### SHADOW SMOOTHING（ノイズ除去）
*   ザラザラした影をAIのように滑らかに仕上げる設定です。通常はデフォルトのままで問題ありません。

### CURVATURE MAP（曲率マップ）※開発中
*   モデルの「角（エッジ）」を検出します。エッジの剥げや汚れを表現するためのテクスチャを作成できます。

### OUTPUT SETTINGS（出力設定）
*   **出力解像度**: 保存される画像のサイズです。
*   **保存先フォルダ**: 指定しない場合は、元のモデルの近くに `BakedAO` フォルダが作られます。
*   **既存ファイルを上書き**: 同じ名前のファイルを置き換えるかどうかです。

### FINISHING TOUCHES（仕上げ）
*   **隙間の穴埋め (Dilation)**: テクスチャの継ぎ目から色が漏れないように、影を少し外側へ広げます。
*   **影の色**: 影の色を指定します。
*   **ぼかし**: 全体的にふわっとさせたい場合に使います。

## 4. Q&A
**Q: ベイク中にUnityが反応しなくなります。**
A: `Low Resource Mode` をオンにしてみてください。また、非常にポリゴン数が多いモデルの場合は少し時間がかかります。

**Q: 影が真っ黒、または真っ白になります。**
A: `Max Distance` の値を調整してみてください。モデルの大きさに合わせて変更する必要があります。

**Q: テクスチャの継ぎ目に変な線が出ます。**
A: `Dilation`（隙間の穴埋め）の値を増やしてみてください。
