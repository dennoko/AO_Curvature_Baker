# Fast AO Baker 使い方ガイド

## 1. ツール概要
Fast AO Bakerは、3Dモデルに対して「Ambient Occlusion (AO / 影)」を計算し、テクスチャとして保存するためのUnityエディタ拡張ツールです。
GPUを利用した高速な計算により、モデルの隅に溜まる影を簡単に作成できます。

## 2. 使用する流れ
1.  **モデルの登録**: `ターゲット` セクションに、影を付けたいオブジェクトをドラッグ＆ドロップします。
2.  **設定の調整**: 必要に応じて、影の濃さや解像度などを調整します（基本はそのままでも動作します）。
3.  **ベイク実行**: 下部の `ベイク実行` ボタンを押します。
4.  **確認**: 計算が終わると、モデルのメインテクスチャと同じフォルダ（または指定したフォルダ）にテクスチャが生成されます。

## 3. 各機能の説明
### ターゲット
*   **ターゲットを追加**: ここに登録したオブジェクトに対してテクスチャが作成されます。MeshFilter または SkinnedMeshRenderer が必要です。

### 遮蔽物メッシュ
*   **影を落とす物を追加**: ターゲットに対して影を落とすための追加オブジェクトを登録できます。背景や他の小物などがこれに当たります。

### 基本AO設定
*   **自分自身の影**: 自分自身が自分に落とす影を計算します。
*   **お互いの影**: 登録したターゲット同士が互いに落とす影を計算します。
*   **低リソースモード**: ベイク中にPCが固まる場合はこれをオンにしてください。

### 詳細設定
*   **影の細かさ**: 影のザラつきに影響します。多いほどきれいです。
*   **影の広がり**: どのくらい遠くの物まで影として考慮するかです。

### 影のノイズ除去
*   ザラザラした影をAIのように滑らかに仕上げる設定です。通常はデフォルトのままで問題ありません。


### 出力設定
*   **出力解像度**: 保存される画像のサイズです。
*   **保存先フォルダ**: 指定しない場合は、元のモデルの近くに `BakedAO` フォルダが作られます。
*   **UVチャンネル**: ベイクに使用するUVセットを選択します。「Auto」は利用可能な最初のチャンネルを自動選択します。特定のチャンネル（ライトマップ用UV1など）にベイクしたい場合に明示的に指定できます。メッシュに存在するUVのみが選択肢に表示されます。
*   **既存ファイルを上書き**: 同じ名前のファイルを置き換えるかどうかです。

### 仕上げ設定
*   **隙間の穴埋め**: テクスチャの継ぎ目から色が漏れないように、影を少し外側へ広げます。
*   **影の色**: 影の色を指定します。
*   **ぼかし**: 全体的にふわっとさせたい場合に使います。

## 4. Q&A
**Q: ベイク中にUnityが反応しなくなります。**
A: `低リソースモード` をオンにしてみてください。また、非常にポリゴン数が多いモデルの場合は少し時間がかかります。

**Q: 影が真っ黒、または真っ白になります。**
A: `影の広がり` の値を調整してみてください。モデルの大きさに合わせて変更する必要があります。

**Q: テクスチャの継ぎ目に変な線が出ます。**
A: `隙間の穴埋め` の値を増やしてみてください。
